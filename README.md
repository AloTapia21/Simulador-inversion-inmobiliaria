# 🏢 Simulador de Inversión Inmobiliaria
### Región Metropolitana de Chile

Herramienta para evaluar y comparar inversiones en departamentos vs alternativas financieras (ETF), desarrollada en Python con Streamlit.

## Funcionalidades
- 📊 Dashboard con KPIs en tiempo real (TIR, ROI, CAGR, Cap Rate, Ganancia Neta)
- 📈 Proyección de flujos a 30 años con gráfico de amortización
- 📉 Análisis de sensibilidad a plusvalía y tasa hipotecaria
- 🔁 Comparador de hasta 5 propiedades en paralelo
- 📄 Exportación de reporte en PDF
- 🚀 Escenarios rápidos: Conservador, Base y Optimista
- 📖 Guía de indicadores inmobiliarios

## Tecnologías
- Python · Streamlit · Plotly · ReportLab
